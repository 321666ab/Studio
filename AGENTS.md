# Project Memory

## Development assistance

- At the start of each development task, evaluate its complexity and suitability for delegation.
- For suitable tasks, prioritize invoking the Claude CLI for implementation, investigation, review, or parallel work, especially when it can reduce Codex token usage without compromising correctness, security, or verification.
- In the final response for every development task, report the number of Claude CLI invocations and briefly state the purpose of each invocation. If it was not invoked, report zero and explain why.
